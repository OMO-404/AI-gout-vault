# 尿酸检验报告数据提取

当用户提供体检报告中的肾功能/尿酸相关检查时，提取和分析痛风相关的关键指标。

## 核心任务

从体检报告照片或PDF中提取尿酸及相关指标，按标准化格式输出，便于存入痛风管理档案。

## 提取内容

### 1. 基本信息
- 姓名、检查日期、医院/体检机构名称
- 检查类型（常规体检/复查/专项检查）

### 2. 核心指标（必须提取）

**尿酸相关**
| 指标 | 说明 |
|------|------|
| 尿酸（UA） | 痛风核心指标，参考值：男性 208-428 μmol/L，女性 155-357 μmol/L |

**肾功能指标**
| 指标 | 说明 |
|------|------|
| 肌酐（Cr） | 评估肾功能，参考值：57-111 μmol/L |
| 尿素氮（BUN） | 肾功能指标，参考值：3.2-7.1 mmol/L |
| eGFR | 估算肾小球滤过率，参考值：>90 mL/min/1.73m² |
| 尿蛋白 | 肾脏损伤指标，参考值：阴性 |
| 尿微量白蛋白 | 早期肾损伤指标，参考值：<30 mg/L |

**肝功能指标**
| 指标 | 说明 |
|------|------|
| ALT（谷丙转氨酶） | 肝功能，参考值：9-50 U/L |
| AST（谷草转氨酶） | 肝功能，参考值：15-40 U/L |

**代谢指标**
| 指标 | 说明 |
|------|------|
| 空腹血糖 | 代谢综合征参考，参考值：3.9-6.1 mmol/L |
| 总胆固醇 | 心血管风险，参考值：<5.18 mmol/L |
| 甘油三酯 | 心血管风险，参考值：<1.7 mmol/L |
| 血尿酸 | 同尿酸（UA） |

## 输出格式

### 主表格

| 项目 | 结果 | 参考范围 | 状态 | 趋势 |
|------|------|----------|------|------|
| 尿酸（UA） | [值] μmol/L | 208-428 | 🔴 偏高 | ↑ |
| 肌酐（Cr） | [值] μmol/L | 57-111 | ✅ 正常 | → |
| eGFR | [值] mL/min | >90 | ✅ 正常 | → |
| 尿蛋白 | [结果] | 阴性 | ⚠️ 弱阳性 | — |

**状态标注**：
- ✅ 正常：在参考范围内
- ⚠️ 偏高/偏低：轻微超出参考范围
- 🔴 异常：明显异常，需要关注

**趋势标注**：
- ↑：比上次升高
- ↓：比上次降低
- →：稳定
- —：无历史数据

### 痛风风险分级

基于尿酸水平，给出风险等级：

| 尿酸水平（μmol/L） | 风险等级 | 说明 |
|-------------------|----------|------|
| <420 | ✅ 低风险 | 正常范围 |
| 420-480 | ⚠️ 中风险 | 高尿酸血症，需关注 |
| 480-540 | 🔴 高风险 | 痛风发作风险高 |
| >540 | 🔴 极高风险 | 需要降尿酸治疗 |

## 专业分析

### 🔍 需要关注的异常项

列出所有异常指标并解释：

**示例**：
- **尿酸偏高**：520 μmol/L，超出正常范围。高尿酸血症是痛风的基础，长期可导致尿酸结晶沉积。
- **肌酐偏高**：可能提示肾功能受损，需结合 eGFR 综合评估。

### 📋 复查建议

- **尿酸偏高**：建议 2-4 周后复查，排除一过性升高
- **肾功能异常**：建议 1-2 周后复查，必要时肾内科就诊
- **尿蛋白阳性**：建议 24 小时尿蛋白定量检查

### 💡 生活方式建议

基于检查结果给出建议：
- 饮食控制（低嘌呤饮食）
- 饮水建议（每日 2000ml 以上）
- 限酒（尤其是啤酒）
- 控制体重

## 触发场景

- 用户上传体检报告图片或 PDF
- 用户说"帮我看看尿酸"、"分析肾功能"、"看看体检结果"
- 用户提供多次检验数据要求对比分析
- 用户询问"我的尿酸高吗"

## 注意事项

- 尿酸水平受饮食、饮酒、药物等因素影响，可能存在波动
- 单次尿酸升高不能确诊痛风，需要结合临床症状
- 痛风发作期尿酸可能正常，需结合发作史综合判断
- 肾功能指标与用药选择相关（如别嘌醇、非布司他需根据肾功能调整剂量）
