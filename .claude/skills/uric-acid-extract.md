# 尿酸检验报告提取

从体检报告照片或 PDF 中提取痛风相关的关键指标，生成结构化分析报告。

## 核心指标映射

| 指标 | 英文 | 参考范围 | 痛风相关性 |
|------|------|----------|------------|
| 尿酸 | UA | 208-428 μmol/L | 🔴 核心 |
| 肌酐 | Cr | 57-111 μmol/L | 用药调整 |
| 尿素氮 | BUN | 3.2-7.1 mmol/L | 肾功能 |
| eGFR | — | >90 mL/min | 用药调整 |

## 风险分级算法

```
if (尿酸 < 420)        → ✅ 低风险
if (420 <= 尿酸 < 480) → ⚠️ 中风险 (高尿酸血症)
if (480 <= 尿酸 < 540) → 🔴 高风险 (发作风险高)
if (尿酸 >= 540)       → 🔴 极高风险 (需要治疗)
```

## 输出格式规范

### 主表格格式

```markdown
| 项目 | 结果 | 参考范围 | 状态 | 趋势 |
|------|------|----------|------|------|
| 尿酸（UA） | [值] μmol/L | 208-428 | 🔴 偏高 | ↑ |
```

**状态标注规则**：
- `✅ 正常`：在参考范围内
- `⚠️ 偏高/偏低`：轻微超出
- `🔴 异常`：明显异常

**趋势标注规则**：
- `↑`：比上次升高
- `↓`：比上次降低
- `→`：稳定
- `—`：无历史数据

## 治疗目标参考

## 注意事项

- 尿酸受饮食、饮酒、药物影响，可能存在波动
- 单次尿酸升高不能确诊痛风，需结合临床症状
- 发作期尿酸可能正常，需结合发作史综合判断
- 肾功能指标影响用药选择（别嘌醇需根据肾功能调整剂量）
