# 痛风用药指导

提供痛风相关药物的用药信息、使用指导和注意事项。

## 核心任务

介绍痛风治疗药物的作用、用法、副作用和注意事项，帮助用户正确使用药物。

## 药物分类

### 1. 急性发作期药物（止痛消炎）

#### 秋水仙碱（Colchicine）

| 项目 | 说明 |
|------|------|
| **作用** | 抑制炎症反应，缓解痛风急性发作 |
| **用法** | 发作初期服用，首剂1mg，1小时后追加0.5mg，12小时后0.5mg每日1-2次 |
| **起效时间** | 通常12-24小时起效 |
| **常见副作用** | 腹泻、恶心、呕吐 |
| **注意事项** | 肝肾功能不全者需减量；过量可致死，严格按医嘱使用 |

#### 非甾体抗炎药（NSAIDs）

| 药物 | 作用 | 用法 | 注意事项 |
|------|------|------|----------|
| **依托考昔** | 选择性COX-2抑制剂 | 120mg 每日1次，急性期用8天 | 胃肠道反应较少 |
| **塞来昔布** | 选择性COX-2抑制剂 | 200mg 每日1-2次 | 磺胺过敏者禁用 |
| **双氯芬酸钠** | 传统NSAID | 50mg 每日2-3次 | 胃肠道刺激，饭后服 |
| **布洛芬** | 传统NSAID | 300-400mg 每日3-4次 | 胃肠道刺激 |
| **吲哚美辛** | 强效NSAID | 25-50mg 每日3次 | 副作用较多 |

**NSAIDs 共同注意事项**：
- 有活动性消化道溃疡者禁用
- 肾功能不全者慎用
- 与抗凝药合用需谨慎
- 建议饭后服用减少胃刺激

#### 糖皮质激素

| 药物 | 用法 | 适用情况 |
|------|------|----------|
| **泼尼松** | 30-40mg/日，3-5天逐渐减量 | NSAIDs禁忌或无效时 |
| **地塞米松** | 0.5-1mg 肌注或口服 | 急性发作快速缓解 |

**注意事项**：
- 不宜长期使用
- 糖尿病患者慎用
- 需要逐渐减量停药

### 2. 降尿酸药物（长期控制）

#### 别嘌醇（Allopurinol）

| 项目 | 说明 |
|------|------|
| **作用** | 抑制尿酸生成，降低血尿酸 |
| **起始剂量** | 100mg 每日1次 |
| **目标剂量** | 300mg 每日1次（可根据尿酸调整至最大600mg/日） |
| **达标时间** | 通常3-6个月 |
| **常见副作用** | 皮疹、胃肠道反应、肝功能异常 |
| **严重风险** | ⚠️ 过敏性休克、重症药疹（HLA-B*5801基因相关） |
| **特殊人群** | 肾功能不全者需减量：eGFR<30ml/min禁用或极低剂量 |

**重要提示**：
- 亚洲人群需警惕 HLA-B*5801 等位基因相关的严重皮疹
- 建议用药前进行基因检测
- 用药期间出现皮疹应立即停药就医

#### 非布司他（Febuxostat）

| 项目 | 说明 |
|------|------|
| **作用** | 抑制尿酸生成，效果强于别嘌醇 |
| **起始剂量** | 40mg 每日1次 |
| **目标剂量**： | 40-80mg 每日1次 |
| **达标时间** | 通常2-4周 |
| **常见副作用** | 肝功能异常、恶心、关节痛（溶晶） |
| **优势** | 肾功能不全者无需调整剂量 |
| **注意事项** | 心血管疾病史患者慎用 |

#### 苯溴马隆（Benzbromarone）

| 项目 | 说明 |
|------|------|
| **作用** | 促进尿酸排泄 |
| **剂量** | 50-100mg 每日1次 |
| **适用人群** | 尿酸排泄减少型患者 |
| **禁忌症** | 肾结石（尿酸性）、重度肾功能不全 |
| **副作用** | 胃肠道反应、肝功能异常 |
| **注意事项** | 用药期间需多饮水，碱化尿液 |

### 3. 辅助药物

#### 碱化尿液药物

| 药物 | 用法 | 目标 |
|------|------|------|
| **碳酸氢钠** | 1g 每日3次 | 尿pH值维持在6.2-6.9 |
| **柠檬酸制剂** | 按说明书 | 同上 |

**作用**：提高尿pH值，增加尿酸溶解度，预防肾结石

## 常见用药问题

### Q1：降尿酸药要吃多久？
**A**：痛风是慢性病，降尿酸药物通常需要长期服用，甚至终身维持。尿酸达标后仍需继续用药维持。

### Q2：尿酸正常了可以停药吗？
**A**：不建议突然停药。突然停药可能导致尿酸反弹，诱发发作。如需停药应在医生指导下逐渐减量。

### Q3：发作时还能吃降尿酸药吗？
**A**：正在服用降尿酸药者，发作期间继续服用，不要停药。尚未开始降尿酸治疗者，建议急性期缓解2-4周后再开始。

### Q4：多种痛风药能一起吃吗？
**A**：
- 降尿酸药通常只用一种（别嘌醇/非布司他/苯溴马隆选其一）
- 急性发作时可加用秋水仙碱或NSAIDs
- 碱化尿液药物可与降尿酸药联用

### Q5：痛风药伤肾吗？
**A**：
- 非布司他对肾功能影响最小
- 别嘌醇需根据肾功能调整剂量
- 苯溴马隆禁用于肾功能不全者
- 未控制的痛风本身会损伤肾脏，合理用药是保护肾脏
