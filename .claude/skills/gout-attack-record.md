# 痛风发作记录

记录痛风发作的详细信息，分析发作规律和诱因，生成个性化预防建议。

## 架构概览

```
┌─────────────────────────────────────────────────────────────────┐
│  发作记录与分析流程                                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  输入: 发作描述（时间/部位/疼痛/诱因）                            │
│           ↓                                                      │
│  结构化记录 → 归类发作特征                                       │
│           ↓                                                      │
│  历史对比 → 识别发作规律                                         │
│           ↓                                                      │
│  诱因统计 → 生成预防建议                                         │
│           ↓                                                      │
│  输出: 发作记录 + 规律分析 + 预防建议                             │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

## 记录维度

### 发作基本信息

| 字段 | 说明 | 示例 |
|------|------|------|
| 发作日期 | 发作开始的具体日期和时间 | 2026-03-15 凌晨 2 点 |
| 发作部位 | 受累关节/部位 | 左脚第一跖趾关节 |
| 疼痛程度 | 疼痛评分（1-10分） | 8分（剧烈疼痛） |
| 持续时间 | 从发作到完全缓解的时间 | 5天 |

### 疼痛程度参考

| 分数 | 程度 | 影响 |
|------|------|------|
| 1-3分 | 轻度 | 不影响日常活动 |
| 4-6分 | 中度 | 影响活动 |
| 7-8分 | 重度 | 严重影响睡眠和活动 |
| 9-10分 | 极重度 | 无法忍受 |

### 诱因分类

| 诱因类型 | 详细说明 |
|----------|----------|
| 饮食因素 | 高嘌呤食物、浓肉汤、饮酒（尤其啤酒） |
| 环境因素 | 受凉、关节外伤、剧烈运动 |
| 药物因素 | 利尿剂、阿司匹林、免疫抑制剂 |
| 代谢因素 | 突然停药、脱水、饥饿 |
| 其他 | 疲劳、压力、作息不规律 |

## 输出格式

### 发作记录模板

```markdown
## 痛风发作记录 - [日期]

### 基本信息
- **发作时间**：[具体时间]
- **发作部位**：[关节/部位]
- **疼痛程度**：[1-10分]
- **持续时间**：[X天]

### 诱因分析
- **饮食**：[前一天饮食情况]
- **环境**：[受凉、外伤等]
- **其他**：[疲劳、压力等]

### 缓解措施
- **用药**：[药物名称、剂量、效果]
- **物理**：[冰敷、休息等]
- **起效时间**：[用药后X小时缓解]
```

### 发作规律分析

当有多次发作记录时：

| 分析项 | 内容 |
|--------|------|
| 发作频率 | 近1个月X次，近3个月X次 |
| 常见诱因 | 饮酒(X次)、海鲜(X次)、受凉(X次) |
| 部位分布 | 左脚大拇指(X次)、右脚踝(X次) |
| 严重程度变化 | 平均疼痛评分的变化趋势 |

## 触发场景

| 用户输入 | 触发动作 |
|----------|----------|
| "痛风犯了" | 引导记录发作详情 |
| "大拇指疼" | 确认是否为痛风发作 |
| "记录发作" | 结构化记录输入 |
| "为什么会发作" | 分析历史诱因 |

## 注意事项

- 痛风发作时尿酸水平可能正常，不要以单次尿酸正常否认诊断
- 首次发作建议就医确诊，排除其他关节疾病
- 发作期间不要随意开始或停用降尿酸药物
- 记录越详细，越有助于发现个人诱因和规律
