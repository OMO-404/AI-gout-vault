# 痛风发作记录与分析

记录痛风发作的详细情况，分析发作规律和诱因。

## 核心任务

记录痛风发作的详细信息，包括发作部位、疼痛程度、持续时间、可能的诱因等，并分析发作规律。

## 记录维度

### 1. 发作基本信息

| 字段 | 说明 | 示例 |
|------|------|------|
| 发作日期 | 发作开始的具体日期和时间 | 2026-03-15 凌晨 2 点 |
| 发作部位 | 受累关节/部位 | 左脚第一跖趾关节（大拇指） |
| 疼痛程度 | 疼痛评分（1-10分） | 8分（剧烈疼痛） |
| 持续时间 | 从发作到完全缓解的时间 | 5天 |

**疼痛程度评分参考**：
- 1-3分：轻度疼痛，不影响日常活动
- 4-6分：中度疼痛，影响活动
- 7-8分：重度疼痛，严重影响睡眠和活动
- 9-10分：极重度疼痛，无法忍受

### 2. 可能诱因

记录发作前 24-72 小时内的情况：

| 诱因类型 | 详细说明 |
|----------|----------|
| **饮食因素** | 高嘌呤食物（海鲜、内脏）、浓肉汤、饮酒（尤其啤酒） |
| **环境因素** | 受凉、关节外伤、剧烈运动 |
| **药物因素** | 利尿剂、阿司匹林、免疫抑制剂等 |
| **代谢因素** | 突然停用降尿酸药、脱水、饥饿 |
| **其他** | 疲劳、压力、作息不规律 |

### 3. 缓解措施

| 措施类型 | 详细说明 |
|----------|----------|
| **用药情况** | 药物名称、剂量、起效时间 |
| **物理措施** | 冰敷、抬高患肢、休息 |
| **其他** | 饮水、碱化尿液等 |

### 4. 发作特征

| 特征 | 说明 |
|------|------|
| 起病方式 | 突然发作/逐渐加重 |
| 疼痛特点 | 胀痛/刺痛/跳痛/烧灼痛 |
| 伴随症状 | 红肿、发热、活动受限 |
| 加重因素 | 行走、触摸、受压 |
| 缓解因素 | 休息、抬高、冰敷 |

## 输出格式

### 发作记录模板

```markdown
## 痛风发作记录 - [日期]

### 基本信息
- **发作时间**：[具体时间]
- **发作部位**：[关节/部位]
- **疼痛程度**：[1-10分]
- **持续时间**：[X天]

### 诱因分析
- **饮食**：[如：前一天晚上吃了海鲜火锅 + 喝啤酒]
- **环境**：[如：天气转凉，脚部受凉]
- **其他**：[如：工作压力大，睡眠不足]

### 缓解措施
- **用药**：[药物名称、剂量、效果]
- **物理**：[冰敷、休息等]
- **起效时间**：[用药后X小时缓解]

### 本次总结
- [本次发作的特点和规律总结]
- [需要注意的诱因]
```

### 📋 个性化预防建议

基于发作历史分析：

**饮食注意事项**
- 明确需要避免的食物（基于个人发作史）
- 提供安全的替代选择

**生活方式调整**
- 针对个人诱因的预防措施
- 作息和运动建议

**用药建议**
- 预防性用药的可能性（需医生评估）
- 症状发作时的备用药物

## 触发场景

- 用户出现痛风发作，需要记录
- 用户说"痛风犯了"、"大拇指疼"、"又发作了"
- 用户要求分析发作规律
- 用户询问"为什么会发作"

## 注意事项

- 痛风发作时尿酸水平可能正常，不要以单次尿酸正常否认痛风诊断
- 首次发作建议就医确诊，排除其他关节疾病
- 发作期间不要随意开始或停用降尿酸药物
- 记录越详细，越有助于发现个人诱因和规律
