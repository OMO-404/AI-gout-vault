# 尿酸趋势与痛风发作关联分析

分析用户的尿酸变化趋势、发作规律，以及两者之间的关联关系，评估治疗效果。

## 核心任务

基于用户的尿酸监测数据和发作记录，进行综合趋势分析，识别风险因素和治疗效果。

## 分析维度

### 1. 尿酸趋势分析

**趋势分类**

| 趋势类型 | 说明 | 判断标准 |
|----------|------|----------|
| ✅ 持续改善 | 尿酸水平稳定下降 | 连续3次检测呈下降趋势 |
| 🔴 持续恶化 | 尿酸水平持续上升 | 连续3次检测呈上升趋势 |
| ⚠️ 波动较大 | 尿酸水平不稳定 | 波动超过参考范围20% |
| 📊 稳定控制 | 尿酸维持在目标水平 | 持续达标（<360 μmol/L） |

**治疗目标评估**
- **一般痛风患者**：尿酸 < 360 μmol/L（6 mg/dL）
- **严重痛风患者**：尿酸 < 300 μmol/L（5 mg/dL）
- **有痛风石患者**：尿酸 < 240 μmol/L（4 mg/dL）

### 2. 发作频率分析

| 分析项 | 说明 |
|--------|------|
| 发作间隔 | 两次发作之间的平均天数 |
| 发作频率 | 每月/每季度/每年的发作次数 |
| 严重程度 | 平均疼痛评分的变化 |
| 持续时间 | 发作持续时间的趋势 |

### 3. 尿酸与发作关联分析

**关联模式识别**

| 模式 | 说明 |
|------|------|
| 高尿酸发作 | 发作时尿酸升高（典型模式） |
| 正常尿酸发作 | 发作时尿酸正常（可能因炎症、药物影响） |
| 降尿酸期发作 | 开始降尿酸治疗后频繁发作（溶晶现象） |
| 无明显关联 | 发作与尿酸水平无明显相关 |

### 4. 诱因统计分析

## 特殊情况说明

### 溶晶现象（降尿酸治疗期间发作增多）

当用户开始降尿酸治疗后，可能出现发作频率增加的情况：

**原因**：血尿酸快速下降时，关节内尿酸盐结晶溶解，刺激关节诱发炎症

**处理建议**：
- 不要因此停用降尿酸药物
- 可小剂量使用秋水仙碱预防发作
- 一般持续 1-3 个月，待尿酸稳定后改善

### 尿酸正常但仍发作

可能原因：
- 急性发作时尿酸可能正常（炎症、药物影响）
- 其他因素诱发（受凉、外伤、疲劳）
- 需要结合发作史综合判断

## 触发场景

- 用户提供多次尿酸检测结果要求分析
- 用户询问"我的尿酸趋势怎么样"
- 用户要求分析发作规律
- 评估治疗效果
- 准备就医，需要了解整体情况

## 注意事项

- 单次尿酸水平不能完全反映控制情况，趋势更重要
- 发作频率和严重程度是评估治疗效果的重要指标
- 降尿酸治疗需要长期坚持，不要短期频繁换药
- 分析结果仅供参考，用药调整请遵医嘱
