# 尿酸趋势分析

基于尿酸监测数据和发作记录，进行综合趋势分析，评估治疗效果并预测达标时间。

## 架构概览

```
┌─────────────────────────────────────────────────────────────────┐
│  趋势分析流程                                                    │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  输入: 多次尿酸检测 + 发作记录                                    │
│           ↓                                                      │
│  尿酸趋势判定 → 改善/恶化/波动/稳定                               │
│           ↓                                                      │
│  发作频率分析 → 发作间隔与严重程度变化                             │
│           ↓                                                      │
│  关联模式识别 → 高尿酸发作/正常尿酸发作/溶晶现象                   │
│           ↓                                                      │
│  治疗效果评估 → 预测达标时间 + 调整建议                           │
│           ↓                                                      │
│  输出: 趋势报告 + 效果评估 + 个性化建议                            │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

## 趋势分类标准

| 趋势类型 | 判断标准 | 标识 |
|----------|----------|------|
| 持续改善 | 连续3次检测呈下降趋势 | ✅ |
| 持续恶化 | 连续3次检测呈上升趋势 | 🔴 |
| 波动较大 | 波动超过参考范围20% | ⚠️ |
| 稳定控制 | 持续达标（<360 μmol/L） | 📊 |

## 治疗目标

| 患者类型 | 目标尿酸 | mg/dL |
|----------|----------|-------|
| 一般痛风患者 | <360 μmol/L | <6 |
| 严重痛风患者 | <300 μmol/L | <5 |
| 有痛风石患者 | <240 μmol/L | <4 |

## 关联模式识别

| 模式 | 说明 | 处理 |
|------|------|------|
| 高尿酸发作 | 发作时尿酸升高 | 典型模式，加强降尿酸 |
| 正常尿酸发作 | 发作时尿酸正常 | 可能是炎症/药物影响 |
| 溶晶发作 | 降尿酸期发作增多 | 不要停药，可加用秋水仙碱 |
| 无明显关联 | 发作与尿酸无关 | 查找其他诱因 |

## 输出格式

### 趋势总览

```markdown
## 尿酸趋势分析

### 检测历史
| 日期 | 尿酸（μmol/L） | 变化 | 达标状态 |
|------|----------------|------|----------|
| 2026-01-15 | 520 | — | 🔴 未达标 |
| 2026-02-15 | 480 | ↓40 | 🔴 未达标 |
| 2026-03-15 | 420 | ↓60 | ⚠️ 接近目标 |

**趋势判定**：✅ 持续改善

### 目标达成情况
- **当前水平**：420 μmol/L
- **目标水平**：<360 μmol/L
- **差距**：60 μmol/L
- **预计达标时间**：约 2 个月（按当前下降速度）
```

## 特殊情况处理

### 溶晶现象

降尿酸治疗期间发作增多，说明：
- 关节内尿酸盐结晶正在溶解
- 是治疗有效的表现，不要停药
- 可小剂量使用秋水仙碱预防
- 一般持续 1-3 个月

## 注意事项

- 单次尿酸水平不能完全反映控制情况，趋势更重要
- 发作频率和严重程度是评估治疗效果的重要指标
- 降尿酸治疗需要长期坚持，不要短期频繁换药
- 分析结果仅供参考，用药调整请遵医嘱
