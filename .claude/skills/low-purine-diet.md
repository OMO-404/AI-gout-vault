# 低嘌呤饮食建议

分析食物嘌呤含量，为痛风患者提供安全的饮食选择和替代方案。

## 架构概览

```
┌─────────────────────────────────────────────────────────────────┐
│  饮食建议流程                                                    │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  输入: 食物名称 / 饮食咨询                                       │
│           ↓                                                      │
│  嘌呤含量查询 → 高/中/低分级                                     │
│           ↓                                                      │
│  个人诱因匹配 → 基于发作史识别风险食物                             │
│           ↓                                                      │
│  替代方案生成 → 推荐低嘌呤替代品                                  │
│           ↓                                                      │
│  输出: 食物评估 + 食用建议 + 替代方案                             │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

## 嘌呤含量分级

| 等级 | 含量范围 | 标识 | 建议 |
|------|----------|------|------|
| 高嘌呤 | >150mg/100g | 🔴 | 尽量避免 |
| 中嘌呤 | 50-150mg/100g | ⚠️ | 适量限制 |
| 低嘌呤 | <50mg/100g | ✅ | 可以食用 |

## 常见食物分类

### 高嘌呤食物（🔴 避免）

| 类别 | 具体食物 |
|------|----------|
| 动物内脏 | 肝、肾、脑、肠、心 |
| 海鲜 | 贝类、沙丁鱼、凤尾鱼、鲭鱼 |
| 浓汤 | 浓肉汤、火锅汤、鸡精 |
| 其他 | 鹅肉、野鸡、酵母粉 |

### 低嘌呤食物（✅ 推荐）

| 类别 | 具体食物 |
|------|----------|
| 主食 | 米饭、面条、面包、玉米、红薯 |
| 蛋奶 | 鸡蛋、牛奶、酸奶、奶酪 |
| 蔬菜 | 白菜、萝卜、黄瓜、番茄、冬瓜 |
| 水果 | 所有水果 |
| 其他 | 海蜇皮、海参、植物油 |

## 输出格式

### 食物查询

```markdown
## [食物名称]

- **嘌呤等级**：🔴 高 / ⚠️ 中 / ✅ 低
- **嘌呤含量**：约 XX mg/100g
- **食用建议**：[能否食用及注意事项]
- **替代食物**：[推荐的低嘌呤替代品]
```

### 个性化建议

基于发作历史的个人风险：
- **🔴 需要避免的食物**（基于个人发作史）
- **✅ 推荐的食物**
- **⚠️ 适量食用的食物**

## 触发场景

| 用户输入 | 触发动作 |
|----------|----------|
| "海鲜能吃吗" | 查询嘌呤含量 |
| "火锅能吃吗" | 评估风险 + 建议 |
| "痛风吃什么" | 提供饮食方案 |
| "发作期能吃什么" | 严格饮食建议 |

## 注意事项

- 发作期需要严格控制（嘌呤<150mg/天）
- 豆制品嘌呤含量中等，发作期避免，缓解期适量
- 充足饮水（>2000ml/天）有助于尿酸排泄
- 避免酒精，尤其是啤酒
