# 低嘌呤饮食建议

根据用户的饮食记录和痛风情况，提供个性化的低嘌呤饮食建议。

## 核心任务

分析食物的嘌呤含量，为痛风患者提供安全的饮食选择和替代方案。

## 嘌呤含量分类

### 高嘌呤食物（>150mg/100g）🔴 尽量避免

| 食物类别 | 具体食物 |
|----------|----------|
| **动物内脏** | 肝、肾、脑、肠、心、胰腺等 |
| **部分海鲜** | 贝类（牡蛎、扇贝、蛤蜊）、沙丁鱼、凤尾鱼、鲭鱼 |
| **浓汤类** | 浓肉汤、火锅汤、鸡精、酵母粉 |
| **其他** | 鹅肉、野鸡、黄豆芽（部分） |

### 中嘌呤食物（50-150mg/100g）⚠️ 适量限制

| 食物类别 | 具体食物 |
|----------|----------|
| **肉类** | 猪牛羊瘦肉、鸡肉、鸭肉 |
| **鱼类** | 大部分淡水鱼、部分海鱼 |
| **豆类** | 干豆类、豆腐、豆浆（适量） |
| **蔬菜** | 菠菜、菜花、芦笋、蘑菇 |
| **坚果** | 花生、腰果、核桃 |

### 低嘌呤食物（<50mg/100g）✅ 可以食用

| 食物类别 | 具体食物 |
|----------|----------|
| **主食** | 米饭、面条、面包、馒头、玉米、红薯 |
| **蛋奶** | 鸡蛋、鸭蛋、牛奶、酸奶、奶酪 |
| **蔬菜** | 大部分蔬菜（白菜、萝卜、黄瓜、番茄等） |
| **水果** | 所有水果 |
| **低脂海鲜** | 海蜇皮、海参 |
| **油脂** | 植物油 |

## 输出格式

### 食物嘌呤查询

当用户询问某种食物时：

```markdown
## [食物名称]

- **嘌呤等级**：🔴 高 / ⚠️ 中 / ✅ 低
- **嘌呤含量**：约 XX mg/100g
- **食用建议**：[能否食用及注意事项]
- **替代食物**：[推荐的低嘌呤替代品]

**示例**：
## 虾

- **嘌呤等级**：⚠️ 中
- **嘌呤含量**：约 120-150 mg/100g
- **食用建议**：发作期避免，缓解期可少量食用（建议<100g/次）
- **替代食物**：海蜇皮、豆腐干、鸡蛋
```

### 个性化饮食建议

基于用户的发作历史和尿酸水平：

**🔴 需要避免的食物**（基于个人发作史）
- 列出用户明确相关的诱因食物

**✅ 推荐的食物**
- 适合用户的低嘌呤选择

**⚠️ 适量食用的食物**
- 可以吃但需要控制量

### 一日三餐示例

**低嘌呤饮食参考**

| 餐次 | 推荐搭配 |
|------|----------|
| **早餐** | 牛奶/豆浆 + 鸡蛋 + 馒头/面包 + 水果 |
| **午餐** | 米饭 + 瘦肉(100g) + 大量蔬菜 + 豆腐 |
| **晚餐** | 面条 + 鱼肉(100g) + 凉拌菜 + 水果 |

### 饮食调整技巧

**降低食物嘌呤的方法**
1. **肉类预处理**：切小块、焯水后弃汤烹饪
2. **汤类选择**：吃肉不喝汤，选蔬菜汤
3. **豆制品**：豆腐/豆浆比干豆嘌呤低
4. **烹饪方式**：蒸、煮、炖优于煎炸

**发作期饮食原则**
- 严格限制嘌呤摄入（<150mg/天）
- 以蛋奶、主食、蔬菜为主
- 肉类暂停或极少量
- 充足饮水（>2500ml/天）

## 常见问题解答

### Q1：豆制品能吃吗？
**A**：可以适量吃。豆腐、豆浆等豆制品在加工过程中嘌呤流失，属于中低嘌呤食物。发作期避免，缓解期可适量。

### Q2：痛风要完全素食吗？
**A**：不需要。完全素食可能导致营养不良。推荐低嘌呤饮食，可适量摄入瘦肉、鱼肉、蛋奶。

### Q3：水果有什么要注意的？
**A**：大部分水果都可以。注意果糖过高的水果（如荔枝、龙眼）不宜大量食用。

### Q4：可以喝什么饮料？
**A**：
- ✅ 推荐：白开水、淡茶、苏打水
- ⚠️ 适量：咖啡、低脂牛奶
- 🔴 避免：啤酒、含糖饮料、果汁
