# 痛风就医报告生成

整理痛风管理档案，生成结构化的就医报告和问题清单，便于与医生高效沟通。

## 架构概览

```
┌─────────────────────────────────────────────────────────────────┐
│  就医报告生成流程                                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  输入: 就医科室/就医意图                                          │
│           ↓                                                      │
│  档案读取 → 尿酸历史/发作记录/用药情况                             │
│           ↓                                                      │
│  数据整理 → 结构化摘要 + 趋势分析                                  │
│           ↓                                                      │
│  问题生成 → 基于个人情况生成针对性问题                             │
│           ↓                                                      │
│  输出: 就医报告（医生版）+ 备忘清单（个人版）                      │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

## 报告结构

### 医生版报告

| 板块 | 内容 |
|------|------|
| 基本信息 | 姓名、年龄、痛风病程 |
| 尿酸历史 | 检测记录 + 趋势图 |
| 发作历史 | 发作频率 + 诱因统计 |
| 当前用药 | 药物名称 + 剂量 + 效果 |
| 重点关注 | 需要讨论的问题 |

### 个人版备忘

| 板块 | 内容 |
|------|------|
| 带材清单 | 报告、化验单、用药清单 |
| 问题清单 | 要问医生的问题 |
| 就医记录 | 医生诊断、调整方案、复查时间 |

## 不同科室重点

| 科室 | 重点关注 | 常见问题 |
|------|----------|----------|
| 风湿免疫科 | 痛风诊断、治疗方案 | 药物选择、剂量调整 |
| 内分泌科 | 代谢综合征 | 血糖、血脂、血压管理 |
| 肾内科 | 肾功能评估 | 肾功能不全时用药调整 |
| 营养科 | 个性化饮食 | 食物选择、膳食搭配 |

## 输出格式

### 就医报告模板

```markdown
# 痛风就医报告

## 基本信息
- **就诊日期**：[日期]
- **痛风病程**：首次发作于[年份]，共发作[X]次

## 尿酸监测历史
| 日期 | 尿酸（μmol/L） | 趋势 | 备注 |
|------|----------------|------|------|
| ... | ... | ... | ... |

## 发作历史
- **最近发作**：[日期]，[部位]，疼痛[评分]分
- **发作频率**：近[X]个月发作[X]次
- **常见诱因**：[诱因列表]

## 当前用药
1. [药物名称] [剂量] [频次]

## 需要医生关注的问题
1. [个性化问题1]
2. [个性化问题2]
3. [个性化问题3]
```

## 问题生成原则

| 类型 | 说明 | 示例 |
|------|------|------|
| ❌ 泛泛问题 | 太宽泛，无针对性 | "痛风怎么治？" |
| ✅ 个性化问题 | 基于个人数据 | "尿酸一直在420，非布司他40mg吃了3个月，需要加量到80mg吗？" |

## 触发场景

| 用户输入 | 触发动作 |
|----------|----------|
| "要去看医生" | 生成就医报告 |
| "要看风湿科" | 针对性准备问题 |
| "就医准备" | 生成准备清单 |
| "要问医生什么" | 生成个性化问题 |

## 注意事项

- 就医报告应信息准确、数据真实
- 问题清单要具体、有针对性
- 就医后及时更新档案信息
- 报告内容仅供就医参考，不替代医生诊断
