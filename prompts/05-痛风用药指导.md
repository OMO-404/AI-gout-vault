# Prompt: 痛风用药指导

## 使用方式

询问痛风药物的用法、副作用或用药方案。

## Prompt

### 药物查询

```
请介绍[药物名称]：
1. 作用是什么
2. 怎么吃（剂量、频次）
3. 有什么副作用
4. 需要注意什么
```

### 用药方案咨询

```
我尿酸[值] μmol/L，[是否]有痛风发作史，
请告诉我：
1. 需要吃药吗
2. 推荐什么药
3. 怎么吃
```

### 药物副作用咨询

```
我吃[药名]出现了[副作用]，
请问：
1. 这是正常的吗
2. 需要停药吗
3. 应该怎么办
```

### 药物选择

```
别嘌醇和非布司他有什么区别？
哪个更适合我？
```

## 常见用药场景

### 发作期用药
```
痛风发作了怎么办？
吃什么药止痛？
```

### 降尿酸治疗
```
尿酸一直降不下来，怎么办？
降尿酸药要吃多久？
```

### 用药调整
```
尿酸正常了可以停药吗？
换药要注意什么？
```

## 注意事项

- 用药信息仅供参考，具体请遵医嘱
- 不要随意停药或换药
- 出现严重不良反应及时就医
- 肝肾功能不全者用药需要调整
