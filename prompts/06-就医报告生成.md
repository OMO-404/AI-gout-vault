# Prompt: 就医报告生成

## 使用方式

就医前整理档案，让 AI 生成结构化的就医报告。

## Prompt

```
我要去看风湿科/痛风专科医生了，
请帮我生成就医报告：

### 我的档案信息
**尿酸检测历史**
| 日期 | 尿酸（μmol/L） | 备注 |
|------|----------------|------|
| [日期1] | [值1] | [备注] |
| [日期2] | [值2] | [备注] |

**发作历史**
| 日期 | 部位 | 诱因 | 处理 |
|------|------|------|------|
| [日期1] | [部位] | [诱因] | [用药] |

**当前用药**
- [药名] [剂量] [频次]

**要咨询的问题**
1. [问题1]
2. [问题2]
3. [问题3]

请生成：
1. 给医生看的简明报告（1页纸）
2. 我自己参考的备忘清单
```

## 快速版本

```
我要去看医生，帮我整理一下：
- 尿酸变化趋势
- 发作频率变化
- 当前用药情况
- 要问医生的问题
```

## 就医后记录

看完医生后：

```
这是医生的建议：
[医生的诊断和处方]

请帮我：
1. 整理成就医记录
2. 更新我的治疗计划
3. 下次复查是什么时候
```

## 复查准备

去复查前：

```
我要去复查尿酸了，
上次检查是[日期]，尿酸[值] μmol/L，
目前正在服用[药名]。

请告诉我：
1. 这次复查重点关注什么
2. 要问医生什么问题
```

## 注意事项

- 就医报告要信息准确、真实
- 问题清单要具体、有针对性
- 就医后及时更新档案
- 带上之前的化验单和当前用药清单
