# Prompt: 痛风发作记录

## 使用方式

痛风发作时，描述症状发给 AI，让其帮你记录并分析。

## Prompt

```
我痛风发作了，帮我记录这次发作情况：

### 发作信息
- 发作时间：[具体时间]
- 发作部位：[哪个关节]
- 疼痛程度：[1-10分，10分最痛]
- 持续时间：[已持续多久/估计持续多久]

### 可能诱因
发作前24-72小时：
- 饮食：[吃了什么特别的吗]
- 饮酒：[喝酒了吗，什么酒，多少]
- 其他：[受凉、疲劳、外伤等]

### 处理情况
- 用药：[吃了什么药，剂量]
- 效果：[有缓解吗]
- 其他措施：[冰敷、休息等]

请帮我：
1. 总结这次发作的特点
2. 分析可能的诱因
3. 给出缓解建议
4. 告诉我如何记录到档案中
```

## 简化版本

快速记录时使用：

```
记录一次痛风发作：
- 时间：[现在]
- 部位：[哪个关节]
- 疼痛：[1-10分]
- 可能原因：[诱因]
```

## 进阶用法

分析发作规律：

```
这是我最近几次的发作记录：
[提供发作历史]

请分析：
1. 发作频率和规律
2. 常见诱因
3. 如何预防下次发作
```

## 注意事项

- 痛风发作时尿酸可能正常，不要以单次尿酸正常否认诊断
- 发作期间记录越详细，越有助于发现个人诱因
- 如果发作频繁或严重，建议及时就医
