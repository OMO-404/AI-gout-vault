# Prompt: 低嘌呤饮食建议

## 使用方式

询问某种食物是否能吃，或要求提供饮食建议。

## Prompt

### 食物查询

```
[食物名称]的嘌呤含量高吗？痛风能吃吗？
```

### 饮食建议

```
我最近痛风发作了/我尿酸偏高，
请给我提供：
1. 需要避免的食物清单
2. 推荐的低嘌呤食物
3. 一日三餐示例
```

### 食物替代

```
我想吃[高嘌呤食物]，有什么低嘌呤的替代品吗？
```

## 快速查询示例

```
虾能吃吗？
火锅能吃吗？
豆腐能吃吗？
啤酒能喝吗？
```

## 发作期饮食

```
我正在痛风发作期，请告诉我：
1. 严格要避免什么
2. 可以吃什么
3. 一天三餐怎么安排
```

## 外食建议

```
要去聚餐/应酬，请告诉我：
1. 可以点什么菜
2. 要避免什么
3. 有什么注意事项
```

## 注意事项

- 发作期饮食控制更严格
- 缓解期可以适量放宽
- 多喝水有助于尿酸排泄
- 避免酒精，尤其是啤酒
