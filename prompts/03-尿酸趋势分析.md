# Prompt: 尿酸趋势分析

## 使用方式

提供多次尿酸检测结果，让 AI 分析趋势和治疗效果。

## Prompt

```
请帮我分析我的尿酸趋势：

### 检测历史
| 日期 | 尿酸（μmol/L） | 备注 |
|------|----------------|------|
| [日期1] | [值1] | [备注] |
| [日期2] | [值2] | [备注] |
| [日期3] | [值3] | [备注] |

### 当前情况
- 目标尿酸：<360 μmol/L
- 当前用药：[药名 + 剂量]
- 用药时间：[开始用药日期]

请分析：
1. 趋势判定（持续改善/恶化/波动）
2. 治疗效果评估
3. 预计多久能达标
4. 需要调整吗
```

## 进阶用法

结合发作情况分析：

```
### 检测历史
[同上]

### 发作历史
| 日期 | 部位 | 尿酸水平 |
|------|------|----------|
| [日期] | [部位] | [当时尿酸] |

请分析：
1. 尿酸与发作的关系
2. 发作频率变化趋势
3. 当前治疗方案是否有效
4. 有什么改进建议
```

## 复查前分析

准备复查前使用：

```
我上次尿酸是[值] μmol/L，距今已[X]个月，
目前正在服用[药名 + 剂量]。

请告诉我：
1. 这次复查重点关注什么
2. 是否需要调整治疗方案
3. 要问医生什么问题
```

## 注意事项

- 单次尿酸水平不能完全反映控制情况，趋势更重要
- 降尿酸治疗需要长期坚持，不要短期频繁换药
- 开始降尿酸治疗后可能发作增多（溶晶现象），是正常反应
