# Prompt: 尿酸检验报告提取

## 使用方式

拍照或上传体检报告中的肾功能/尿酸检查，把下面的 Prompt 和照片一起发给 AI。

## Prompt

```
请帮我从这份体检报告中提取尿酸及相关指标，按以下格式输出：

### 核心指标
| 项目 | 结果 | 参考范围 | 状态 | 趋势 |
|------|------|----------|------|------|
| 尿酸（UA） | [值] μmol/L | 208-428 | [状态] | [趋势] |
| 肌酐（Cr） | [值] μmol/L | 57-111 | [状态] | [趋势] |
| eGFR | [值] mL/min | >90 | [状态] | [趋势] |
| 尿蛋白 | [结果] | 阴性 | [状态] | — |

如果有我之前的检测结果，请在"趋势"列标注：
- ↑ 升高
- ↓ 降低
- → 稳定
- — 无历史数据

### 风险评估
基于当前尿酸水平，告诉我：
1. 风险等级（低/中/高/极高）
2. 是否达标（目标：<360 μmol/L）
3. 需要注意的异常项

### 建议
给出：
1. 复查建议
2. 生活方式建议
```

## 进阶用法

如果有多次检测结果，追加：

```
这是我最近 3 次的尿酸检测结果：
[提供历史数据]

请分析趋势，告诉我：
1. 尿酸是在改善还是恶化
2. 当前的治疗是否有效
3. 下一步该怎么办
```

## 注意事项

- 拍照时确保文字清晰，光线充足
- 如果有肾功能异常，AI 会提示需要注意用药调整
- 单次尿酸正常不能排除痛风，需要结合发作史综合判断
