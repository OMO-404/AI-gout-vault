# 快速开始：15 分钟搭建痛风管理系统

## 你需要准备的

- [Obsidian](https://obsidian.md/)（免费，全平台）
- Claude Code（或 Claude/ChatGPT/Gemini 任一 AI）
- 最近的尿酸检测报告（如有）

---

## Step 1: 获取模板（2 分钟）

```bash
# 方式一：Git Clone
git clone https://github.com/[你的用户名]/ai-gout-vault.git

# 方式二：下载 ZIP
# 点击 GitHub 页面右上角 Code → Download ZIP
```

---

## Step 2: 用 Obsidian 打开（1 分钟）

1. 打开 Obsidian
2. 点击「打开文件夹作为仓库」
3. 选择 `ai-gout-vault/vault/` 文件夹
4. 你会看到**痛风管理中心**作为首页

---

## Step 3: 填写基本信息（3 分钟）

1. 打开 `尿酸监测档案.md`
2. 填写基本信息（姓名、出生日期、身高等）
3. 如有尿酸检测报告，拍照备用

---

## Step 4: 录入第一次尿酸数据（5 分钟）

1. 打开 `prompts/01-尿酸检验提取.md`
2. 复制里面的 Prompt
3. 把 Prompt + 体检报告照片一起发给 AI
4. AI 输出结构化表格 → 粘贴到 `尿酸监测档案.md`
5. 同时更新 `tracking/尿酸记录.csv`

---

## Step 5: 记录发作史（可选，5 分钟）

如果有痛风发作史：

1. 打开 `prompts/02-痛风发作记录.md`
2. 复制 Prompt，告诉 AI 你的发作情况
3. 将结果记录到 `痛风发作档案.md`
4. 更新 `tracking/发作记录.csv`

---

## 常见问题

### Q: Claude Code 和普通 Claude 有什么区别？

**A**: Claude Code 是 Claude 的 CLI 版本，可以自动加载 `.claude/skills/` 下的技能，使用更方便。普通 Claude/ChatGPT/Gemini 需要手动复制 `prompts/` 下的提示词。

### Q: 数据安全吗？

**A**: 所有数据存储在本地 Obsidian 中，不会自动上传。但发给 AI 分析时，数据会经过 AI 服务商服务器。

### Q: 可以给家人用吗？

**A**: 本版本为个人使用设计。如需多人管理，可复制模板创建多个档案。

### Q: 能替代医生吗？

**A**: 不能。这是辅助管理工具，用药和治疗请遵医嘱。
