## 工作流

### 收到新体检报告/化验单时

```
1. 拍照/PDF 发给 AI
2. AI 提取尿酸及相关数据 → 写入 [[尿酸监测档案]]
3. AI 自动对比历史数据 → 标注趋势变化
4. 如有异常 → AI 分析风险并给出建议
```

### 痛风发作时

```
1. 记录发作详情（时间、部位、疼痛程度等）
2. 分析可能的诱因（饮食、饮酒、受凉等）
3. 记录缓解措施和用药情况
4. 发作缓解后 → 请求 AI 分析发作规律
```

### 饮食咨询时

```
1. 告诉 AI 想吃的食物
2. AI 查询嘌呤含量 → 给出食用建议
3. 如果是高嘌呤食物 → AI 推荐低嘌呤替代品
4. 更新到饮食记录
```

### 就医前准备

```
1. 告诉 AI "我要去看风湿科"
2. AI 读取档案 → 生成就医报告：
   - 尿酸历史趋势
   - 发作历史摘要
   - 当前用药情况
   - 建议追问医生的问题
3. 就医后 → 把医生的结论发给 AI → 更新档案
```

### 复查提醒

```
1. 根据医生建议设置复查日期
2. 到期前 AI 提醒复查
3. 复查后更新尿酸监测档案
```

## 使用提示

### 快捷操作

- **查尿酸**：告诉 AI "帮我看看最新的尿酸报告"
- **记发作**：告诉 AI "记录一次痛风发作"
- **问食物**：告诉 AI "XX能吃吗"
- **准备就医**：告诉 AI "我要去看医生"
- **分析趋势**：告诉 AI "分析我的尿酸趋势"

### 常用命令

| 命令 | 说明 |
|------|------|
| "分析尿酸" | 分析尿酸变化趋势 |
| "记录发作" | 记录痛风发作详情 |
| "XX能吃吗" | 查询食物嘌呤含量 |
| "就医报告" | 生成就医报告 |
| "用药建议" | 查询痛风用药信息 |
