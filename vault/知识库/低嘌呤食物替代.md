# 低嘌呤食物替代方案

> 用低嘌呤食物替代高嘌呤食物，让痛风患者也能吃得丰富多样。

---

## 肉类替代

### 高嘌呤肉类 → 低嘌呤替代

| 原食物 | 嘌呤等级 | 替代食物 | 说明 |
|--------|----------|----------|------|
| 动物内脏（肝、肾等） | 🔴 极高 | 鸡蛋 | 蛋白质丰富，无嘌呤 |
| 肥肉 | 🔴 高 | 瘦肉 | 限量<100g/日 |
| 浓肉汤 | 🔴 极高 | 蔬菜汤 | 嘌呤含量极低 |
| 火锅汤底 | 🔴 极高 | 清汤/番茄汤 | 避免嘌呤高的汤底 |

### 烹饪建议

```markdown
**焯水法**（降低肉类嘌呤）
1. 肉类切小块
2. 沸水焯1-2分钟
3. 弃掉汤汁
4. 再进行正常烹饪
```

## 海鲜替代

### 高嘌呤海鲜 → 低嘌呤替代

| 原食物 | 嘌呤等级 | 替代食物 | 说明 |
|--------|----------|----------|------|
| 贝类（牡蛎、扇贝、蛤蜊） | 🔴 极高 | 海蜇皮 | 口感爽脆，低嘌呤 |
| 沙丁鱼、凤尾鱼 | 🔴 极高 | 鲈鱼（限量） | 中嘌呤，缓解期适量 |
| 浓鱼汤 | 🔴 极高 | 清蒸鱼 | 嘌呤含量较低 |
| 虾/蟹 | ⚠️ 中高 | 豆腐干 | 植物蛋白替代 |

## 豆类替代

### 高嘌呤豆类 → 低嘌呤替代

| 原食物 | 嘌呤等级 | 替代食物 | 说明 |
|--------|----------|----------|------|
| 干黄豆 | 🔴 高 | 豆腐 | 加工后嘌呤流失 |

| 浓豆浆 | ⚠️ 中 | 稀释豆浆 | 适量饮用 |
| 五香豆干 | ⚠️ 中 | 鸡蛋 | 蛋白质替代 |

---

## 蔬菜选择

### 中高嘌呤蔬菜 → 低嘌呤蔬菜

| 原食物 | 嘌呤等级 | 替代蔬菜 |
|--------|----------|----------|
| 菠菜 | ⚠️ 中 | 小白菜、油麦菜 |
| 菜花 | ⚠️ 中 | 白菜、卷心菜 |
| 芦笋 | ⚠️ 中 | 芦笋尖（少量）、豆角 |
| 蘑菇 | ⚠️ 中 | 木耳、银耳 |
| 竹笋 | ⚠️ 中 | 莴笋、冬瓜 |

### 推荐低嘌呤蔬菜

```markdown
**绿叶蔬菜**
- 小白菜、油菜、油麦菜
- 菠菜（发作期限制）

**瓜茄类**
- 冬瓜、黄瓜、南瓜、茄子、番茄

**根茎类**
- 白萝卜、胡萝卜、莲藕、土豆
```

## 一日三餐示例

### 低嘌呤饮食参考

**早餐**
```
✅ 牛奶/豆浆 + 鸡蛋 + 馒头/面包
✅ 小米粥 + 鸡蛋 + 凉拌黄瓜
```

**午餐**
```
✅ 米饭 + 清蒸鱼（100g）+ 大量蔬菜 + 豆腐
✅ 面条 + 鸡丝（100g）+ 凉拌菜
```

**晚餐**
```
✅ 杂粮饭 + 瘦肉（80g）+ 清炒蔬菜
✅ 红薯 + 海蜇皮 + 蔬菜汤
```

## 外食建议

### 外食选择

| 场景 | 推荐选择 | 避免 |
|------|----------|------|
| **中餐** | 清蒸、白灼、炒蔬菜 | 红烧、油炸、火锅 |
| **日料** | 寿司、刺身（限量）、天妇罗 | 鱼子、贝类、炸物 |
| **西餐** | 沙拉、烤鸡、土豆泥 | 肉酱、浓汤、内脏 |
| **快餐** | 鸡肉汉堡（去酱） | 牛肉汉堡、炸鸡 |

## 节假日饮食

### 节假日注意事项

1. **聚餐前**：提前吃点东西垫肚子
2. **聚餐时**：多吃蔬菜，少吃肉，不喝酒
3. **聚餐后**：多喝水，监测尿酸

### 应对策略

```markdown
**遇到高嘌呤食物时**
- 少量尝一口，不贪多
- 优先吃蔬菜和主食
- 饭后大量饮水
```
